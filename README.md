# STAT2003/STAT7003 2026 Semester - UQ

Live demos from class for the STAT2003/STAT7003 course at The University of Queensland, Semester 1 2026.

## Installing Python

- **Mac**: Install via [Homebrew](https://brew.sh/) by running `brew install python`, or download from [python.org](https://www.python.org/downloads/macos/).
- **Windows**: Download from [python.org](https://www.python.org/downloads/windows/). Make sure to check "Add Python to PATH" during installation.

## Running

Clone this repository and run the start script:

```bash
git clone https://github.com/yoninazarathy/stat2003_2026_demos.git
cd stat2003_2026_demos
./start.sh
```

This installs the required packages (`jupyter`, `numpy`, `matplotlib`) and opens Jupyter Notebook in your browser.
